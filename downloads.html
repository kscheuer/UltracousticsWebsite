A note on Windows SmartScreen. The current build is not yet code-signed, so Windows may show a "Windows protected your PC" warning. This is expected for small-publisher software. Installing

  1. Download the installer using the button above.
  2. Verify the SHA-256 checksum matches the value shown here.
  3. Double-click the .exe. If SmartScreen appears, choose More info → Run anyway.
  4. Follow the installer prompts.
  5. Connect your BROADSONIC unit via USB and launch ReadoutGUI.

System Requirements

  • OS: Windows 10 (1809+) or Windows 11, 64-bit
  • CPU: x64 processor, 2 cores or more
  • RAM: 4 GB minimum, 8 GB recommended
  • Disk: ~200 MB free
  • Connectivity: USB for hardware interface
